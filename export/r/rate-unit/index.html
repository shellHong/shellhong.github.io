百分比单位

left、top

  • left：相对于第一个定位父元素的宽度进行百分比换算
  • top：相对于第一个定位父元素的高度进行百分比换算
是第一个定位父元素，而不是第一个父元素

translateX、translateY

  • translateX：相对于元素本身的宽度进行百分比换算
  • translateY：相对于元素本身的高度进行百分比换算

width、height

  • width：相对于其父元素的宽度进行百分比换算
  • height：相对于其父元素的高度进行百分比换算

margin-left、margin-top

  • margin-left：相对于其父元素的宽度进行百分比换算
  • margin-top：相对于其父元素的宽度进行百分比换算
margin相关的百分比值，都是相对于其父元素的宽度进行百分比换算的

padding-left、padding-top

  • padding-left：相对于其父元素的宽度进行百分比换算
  • padding-top：相对于其父元素的宽度进行百分比换算
padding相关的百分比值，和margin一样，都是相对于其父元素的宽度进行百分比换算的

应用场景

  • 高度不确定的元素垂直居中（相对于其定位父元素） 
    div{
  position: absolute;
  top:50%;
  transform: translateY(-50%);
}
  • 宽高成比例的元素（例如1:1为正方形） 
    div{
  width:0;
  height:0;
  padding-top:50%;
  padding-left: 50%;
}
    or 
    div{
  height:0;
  padding-top:50%;
  width: 50%;
}
